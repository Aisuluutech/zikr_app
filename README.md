# Zikr App (Flutter)

## Русский

**Dhikr App** — учебное приложение на Flutter, созданное для практики разработки мобильных приложений.

**Функциональность:**
- Список зикров с переводом  
- Отдельный счётчик для каждого зикра  
- Воспроизведение аудио  
- Дополнительная информация о каждом зикре  
- Возможность добавлять собственные зикры  

**Важно:**  
Дизайн, тексты, переводы и аудио взяты из существующего приложения.  
Проект создан исключительно в учебных целях и не предназначен для коммерческого использования.

---

## English

**Zikr App** is a Flutter learning project created to practice mobile app development.

**Features:**
- List of dhikrs with translations  
- Individual counter for each dhikr  
- Audio playback  
- Additional information about each dhikr  
- Users can add their own dhikrs  

**Important:**  
The design, texts, translations, and audio are taken from an existing application.  
This project is for educational purposes only and not intended for commercial use.
